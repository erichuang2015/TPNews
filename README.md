# TPNews
一个简单的新闻发布系统，thinkphp3.2 + bootstrap

1、将TPNews放入目录下  
2、导入数据库（新建数据库tp_new，字符串为utf-8，导入文件夹下的tp_new.sql文件）  
3、修改Application\Common\Conf\config.php为你的数据库配置  
4、访问 http://127.0.0.1/TPNews/index.php 即可，端口号默认80（若有改动，自行变化）
